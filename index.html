Hola, Soy Fernando López

Ingeniero en Ciencias y Sistemas

Soy un ingeniero de desarrollo especializado en frontend y backend, con habilidades en automatización de despliegues usando GitHub Actions y en la optimización de código HTML con Emmet. Domino el uso de tecnologías web modernas para crear interfaces de usuario eficientes, y también tengo experiencia en configuración de estilos CSS avanzados y SEO. En backend, manejo la integración de Blazor en .NET Core MVC para mejorar la productividad y rendimiento de aplicaciones web, además de un buen conocimiento en la administración de hardware, incluyendo tipos y capacidades de discos duros. Mi enfoque siempre está en brindar soluciones completas y optimizadas tanto en el desarrollo de interfaces como en la lógica del servidor.
